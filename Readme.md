# Predicción de Churn de Clientes de Telecomunicaciones

## Descripción del proyecto

Este proyecto tiene como objetivo predecir la cancelación de clientes (churn) en una empresa de telecomunicaciones mediante técnicas de Machine Learning.

La retención de clientes es un aspecto fundamental para las empresas, ya que adquirir nuevos clientes suele ser más costoso que conservar los existentes. Por ello, se desarrolló un modelo predictivo capaz de identificar a los clientes con mayor probabilidad de cancelar el servicio.

## Objetivo

Construir un modelo de aprendizaje automático que permita predecir si un cliente abandonará la compañía utilizando información sobre contratos, servicios contratados y características de los clientes.

## Datos utilizados

El conjunto de datos incluye información relacionada con:

* Tipo de contrato
* Tiempo de permanencia del cliente
* Servicios de internet
* Servicios telefónicos
* Método de pago
* Facturación mensual
* Facturación total
* Estado de cancelación del servicio (Churn)

## Metodología

1. Exploración y análisis de datos.
2. Limpieza y preparación de datos.
3. Transformación de variables categóricas.
4. División de datos en conjuntos de entrenamiento y prueba.
5. Entrenamiento de diferentes modelos de Machine Learning.
6. Evaluación del desempeño mediante métricas de clasificación.
7. Selección del mejor modelo.

## Herramientas utilizadas

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Jupyter Notebook

## Resultados

Se desarrolló un modelo capaz de identificar clientes con riesgo de abandono, permitiendo apoyar estrategias de retención y toma de decisiones basadas en datos.
